# イベントURL活用ガイド - みんなのWA

## 🔗 イベント専用URLシステム

イベントカードが非常に重要なため、各イベントへ簡単にアクセス・共有できるURL構造を実装しました。

---

## 📋 URLの種類

### 1️⃣ **イベント詳細URL**（SEO対応）

**形式:** `/events/<イベントID>`

**例:**
- `/events/event-001` - 第1回イベント
- `/events/event-002` - 第2回イベント
- `/events/event-006` - 第6回イベント

**特徴:**
- ✅ **OGPタグ完全対応** - SNSシェア時に綺麗なプレビュー表示
- ✅ **Twitter Card対応** - Twitterで魅力的な表示
- ✅ **SEO最適化** - 検索エンジンに最適化
- ✅ **404エラー処理** - 存在しないイベントは適切にエラー表示
- ✅ **自動リダイレクト** - メインページの該当イベントへスムーズに移動

**使用例:**
```
https://3000-i26mzzri6plazftdqznnd-b32ec7bb.sandbox.novita.ai/events/event-001
```

---

### 2️⃣ **直接リンクURL**（パラメータ指定）

**形式:** `/?event=<イベントID>`

**例:**
- `/?event=event-001` - 第1回イベントへジャンプ
- `/?event=event-002` - 第2回イベントへジャンプ
- `/?event=event-006` - 第6回イベントへジャンプ

**特徴:**
- ✅ **該当イベントへ自動ジャンプ** - カルーセルが自動的に該当イベントを表示
- ✅ **スムーズスクロール** - イベントセクションへ自動スクロール
- ✅ **500ms遅延** - ページ読み込み後に確実に実行
- ✅ **シンプルな構造** - クエリパラメータで簡単指定

**使用例:**
```
https://3000-i26mzzri6plazftdqznnd-b32ec7bb.sandbox.novita.ai/?event=event-001
```

## 🎯 使用シーン別活用方法

### 📱 SNSでイベントをシェアする場合

**推奨URL:** `/events/<イベントID>`

**理由:**
- OGPタグによりSNS上で魅力的なプレビュー表示
- イベント画像、タイトル、説明が自動表示
- クリック率が大幅に向上

**例:**
```
https://3000-i26mzzri6plazftdqznnd-b32ec7bb.sandbox.novita.ai/events/event-003
```

→ Facebook、Twitter、LINEなどでシェア時に綺麗なカード表示

---

### 📧 メール・チラシで特定イベントを案内する場合

**推奨URL:** `/events/<イベントID>` または `/?event=<イベントID>`

**どちらも可:**
- `/events/<イベントID>` - よりプロフェッショナルな印象
- `/?event=<イベントID>` - シンプルで短い

**例:**
```
第3回異業種交流会の詳細はこちら:
https://3000-i26mzzri6plazftdqznnd-b32ec7bb.sandbox.novita.ai/events/event-003
```

## 🔄 動作フロー

### `/events/<イベントID>` アクセス時

```
1. サーバーがイベント情報を取得
   ↓
2. OGPタグ付きHTMLページを生成
   ↓
3. 自動的に /?event=<イベントID> へリダイレクト
   ↓
4. メインページで該当イベントへジャンプ＆スクロール
```

### `/?event=<イベントID>` アクセス時

```
1. メインページ読み込み
   ↓
2. URLパラメータを検出
   ↓
3. 500ms後に該当イベントへジャンプ
   ↓
4. イベントセクションへスムーズスクロール
```

## 💡 実装のポイント

### サーバーサイド（server.js）
```javascript
// SEO対応イベント詳細ページ
app.get('/events/:id', (req, res) => {
    // OGPタグ付きHTMLを返却
    // 自動リダイレクト: /?event=<id>
});
```

### クライアントサイド（script.js）
```javascript
// URLパラメータで該当イベントへジャンプ
const urlParams = new URLSearchParams(window.location.search);
const eventId = urlParams.get('event');
if (eventId) {
    jumpToEvent(eventIndex);
    eventSection.scrollIntoView({ behavior: 'smooth' });
}
```
