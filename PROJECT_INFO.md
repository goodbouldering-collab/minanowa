Genspark の有料機能が終了した場合でも、以下の方法でプロジェクトを継続できます：

### 1. ローカル開発環境での実行
```bash
# バックアップファイルを展開
tar -xzf minanowa_backup_YYYYMMDD_HHMMSS.tar.gz

# 依存関係をインストール
npm install

# サーバーを起動
node server.js
```

### 2. 別のホスティングサービスへの移行
- Vercel
- Netlify
- Heroku
- AWS
- さくらのレンタルサーバー

### 3. GitHubからのクローン
```bash
git clone https://github.com/goodbouldering-collab/minanowa.git
cd minanowa
npm install
node server.js
```
