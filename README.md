# みんなのWA - 彦根発 異業種交流コミュニティ

## 🎉 プロジェクト概要

滋賀県彦根市を中心とした異業種交流会のWebサイトです。
月1回の交流会を通じて、地域の事業者同士がつながり、新しいコラボレーションを生み出すプラットフォームを提供します。

## 📝 主要なAPIエンドポイント

### 認証
- `POST /api/register` - 会員登録
- `POST /api/login` - ログイン

### データ取得
- `GET /api/members` - 全メンバー取得
- `GET /api/events` - 全イベント取得
- `GET /api/blogs` - 全ブログ取得

### データ更新
- `PUT /api/members/:id` - メンバー情報更新
- `POST /api/events/:id/register` - イベント参加登録

### 管理者機能
- `POST /api/admin/events` - イベント作成
- `PUT /api/admin/events/:id` - イベント更新
- `DELETE /api/admin/events/:id` - イベント削除
- `POST /api/admin/blogs` - ブログ作成
- `PUT /api/admin/blogs/:id` - ブログ更新
- `DELETE /api/admin/blogs/:id` - ブログ削除
- `GET /api/admin/backup` - データバックアップ

バックアップファイルは `backups/` フォルダに保存されます。
